# 🌟 Scroll-Driven Hero Animation (Assignment Project)

## 📌 Project Overview

This project is a **scroll-driven animated hero section** created as part of a web development assignment. It demonstrates modern frontend animation techniques using smooth motion, parallax effects, and interactive UI elements.

The goal of this assignment was to build a visually engaging hero section that responds to user scrolling instead of time-based animations.

---

## 🎯 Assignment Objective

* Create a full-screen hero section
* Implement scroll-based animations
* Add staggered text reveal effects
* Display animated statistics counters
* Ensure smooth performance using transform animations
* Use GSAP ScrollTrigger for scroll control

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
* GSAP (GreenSock Animation Platform)

---

## ✨ Features

* Premium animated gradient background
* Letter-by-letter heading animation
* Glassmorphism stat cards
* Animated number counters
* Parallax hero image motion
* Scroll-controlled transitions
* Smooth pinned hero section
